Class TokenWalletFilterList

A filterable list of TokenWalletAccounts.

Use the filter methods to select the accounts that you want to interact with.

To filter a subset of accounts, see ForToken, WithSymbol, WithMint, WithAtLeast, WhichAreAssociatedTokenAccounts,

To select individual accounts, see: WithPublicKey, AssociatedTokenAccount
